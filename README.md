# Emotion-Aware Conversational Agent

This project is a final year major project developed as part of the Computer Science and Design curriculum.

## Project Description
The system analyzes human emotions using text, audio, and video inputs. Text-based emotion detection is handled through a chatbot, while audio and video emotions are detected from recorded responses.

## Modules
- Text Emotion Chatbot
- Audio Emotion Detection
- Video Emotion Detection

## Technologies Used
- Python
- TensorFlow / Keras
- OpenCV
- Librosa
- Flask

## How to Run
1. Install required libraries:
   pip install -r requirements.txt
2. Run the application:
   python app.py

## Future Scope
- Multimodal emotion fusion
- Real-time emotion detection
- Multilingual support
